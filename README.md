## 📊 Real Data Sources

| Dataset | Source | Records | Use Case |
|---------|--------|---------|----------|
| **CDC Wonder** | CDC | 3M+ deaths/yr | Mortality trend analysis, opioid epidemic tracking |
| **CMS Medicare Cost Reports** | data.cms.gov | 6,000+ hospitals | Hospital cost efficiency, readmission rates |
| **HHS Health Indicators** | healthdata.gov | 1,200+ metrics | Population health benchmarking |
| **HCUP NIS** | AHRQ | 7M+ discharges/yr | Inpatient utilization patterns |
| **NYC EMS 911** | NYC OpenData | 2M+ calls/yr | Emergency response time analysis |
