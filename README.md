# Sierra Healthcare Analytics

> **Healthcare Data Scientist | Medicaid Utilization · 911 Triage Impact · Public Health Dashboards**

I'm a healthcare data scientist who turns messy Medicaid claims, EMS dispatch logs, and public health surveillance data into decisions that save lives and money.
